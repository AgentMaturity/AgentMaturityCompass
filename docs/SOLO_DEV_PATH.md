# SOLO_DEV_PATH.md — AMC path for solo builders and OSS maintainers

## Who this is for
- solo developers
- indie builders
- open-source maintainers
- early-stage AI app builders

## What you probably want
- a fast trust score
- obvious gaps
- practical fixes
- lightweight CI checks
- no enterprise ceremony on day one

## Best path
1. `docs/START_HERE.md`
2. `docs/QUICKSTART.md`
3. `docs/AFTER_QUICKSCORE.md`
4. `docs/CI_TEMPLATES.md`
5. `docs/EXAMPLES_INDEX.md`

## Recommended product focus
- **Score**
- **Shield**
- light **Watch** usage

## Suggested sequence
1. run `npx agent-maturity-compass quickscore`
2. inspect the gaps
3. run `amc fix`
4. add one CI gate
5. try one framework example

## What to ignore for now
You probably do not need deep enterprise deployment, policy packs, or full compliance reporting on day one.
