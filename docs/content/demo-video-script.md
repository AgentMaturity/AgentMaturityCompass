# AMC Demo Video Script (60-second version)

## Scene 1: The Problem (0-10s)
**Visual:** Terminal, blank screen
**Narration:** "Your AI agent says it's safe. But can it prove it?"

## Scene 2: One Command (10-20s)
**Visual:** Type `npx agent-maturity-compass quickscore --rapid`
**Narration:** "One command. No account. No API key. Agent Maturity Compass scores your agent in seconds."

## Scene 3: The Score (20-30s)
**Visual:** Score output showing L2, dimension breakdown
**Narration:** "L0 to L5 maturity across 5 dimensions. Not pass/fail — a roadmap to production-ready."

## Scene 4: The Fix (30-40s)
**Visual:** Type `amc fix --target-level L3`
**Narration:** "AMC doesn't just score — it generates guardrails, config patches, and CI gates automatically."

## Scene 5: Red Team (40-50s)
**Visual:** Type `amc assurance run --all`, show pack results scrolling
**Narration:** "147 adversarial attack scenarios. Prompt injection. Exfiltration. Alignment faking. All tested."

## Scene 6: Proof (50-55s)
**Visual:** Show signed report SHA
**Narration:** "Every score is cryptographically signed. Tamper-evident. Audit-ready."

## Scene 7: CTA (55-60s)
**Visual:** AMC logo + GitHub URL
**Narration:** "Agent Maturity Compass. Free. Open source. Stop trusting. Start verifying."

# Recording Notes
- Use iTerm with dark theme, large font (JetBrains Mono 18pt)
- Terminal recording: `asciinema rec` or screen record with QuickTime
- Convert to GIF with `gifski` or `asciinema` player embed
- Optimal GIF size: 800x450, <5MB for GitHub README
