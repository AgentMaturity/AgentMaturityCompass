# PRICING_FAQ.md — Commercial questions people will ask

## Is AMC free?
AMC has an open-source core and free entry path.

## What is paid?
Team, managed, enterprise, support, or service value beyond the open-source entry path.

## Is sponsorship the same as buying AMC?
No.
Sponsorship supports the open-source project.
Pricing buys product, deployment, support, or service value.

## Do I need enterprise to try AMC?
No.
Browser and CLI entry paths should stay easy.
