# HOMEPAGE_TRUST_SIGNALS.md — What trust/proof should be surfaced

AMC already has strong technical substance. The problem is packaging it visibly.

## Signals worth surfacing
- test count
- framework breadth
- assurance/pack breadth
- maturity scale clarity
- MIT license
- browser + CLI + CI paths
- support policy / stability expectations

## Rule
Do not fake dynamism.
If counters are manual, keep them honest and reasonably maintained.
