# Skales.app GTM Comparison & Response Plan

## What Skales Is
Skales.app positions as an "AI agent evaluation platform" — web-based, SaaS model. Targets teams evaluating and comparing AI agents for enterprise deployment.

## AMC vs Skales

| Dimension | AMC | Skales |
|---|---|---|
| Delivery | CLI + self-hosted | SaaS platform |
| Pricing | Free/open source (MIT) | SaaS pricing |
| Data residency | 100% local — data never leaves your machine | Cloud-hosted |
| Evidence model | Cryptographic proofs | Platform-managed |
| Framework support | 14 adapters | Limited integrations |
| Customization | Full source access | Configuration-only |
| Compliance | Automated binder generation | Manual reporting |

## GTM Differences
- **Skales targets**: Enterprise buyers who want a dashboard and managed service
- **AMC targets**: Developers and platform teams who want CLI-first, auditable, self-hosted

## Response
- AMC's GTM advantage: `npx` install, no vendor lock-in, data stays local
- For enterprises wanting a dashboard: `amc dashboard open` provides local web UI
- For compliance: AMC generates artifacts that Skales can't (tamper-evident, signed)
- Messaging: "Your data. Your scores. Your proofs. No cloud required."
