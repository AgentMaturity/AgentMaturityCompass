# Zeroboot Competitive Response Plan

## What Zeroboot Is
Zeroboot offers AI agent testing and validation tools. Smaller player focused on automated testing of AI assistants and chatbots.

## Threat Level: Low
Smaller community, narrower scope, less technical depth than AMC.

| Dimension | AMC | Zeroboot |
|---|---|---|
| Scope | Full trust stack (8 products) | Testing only |
| Maturity model | L0-L5 across 5 dimensions | No maturity model |
| Assurance depth | 147 packs, evolutionary red-team | Basic test suites |
| Compliance | EU AI Act, ISO 42001, NIST, SOC 2, OWASP | None |
| Crypto evidence | Ed25519 + Merkle trees | None |
| Domain packs | 40 industry packs | None |
| Framework adapters | 14 | Limited |
| Open source | Yes (MIT) | Partially |

## Response Strategy

- If users ask: position as "AMC does everything Zeroboot does, plus governance, compliance, and cryptographic evidence"
- No need for dedicated comparison page — covered in general comparison

## Monitoring
