# COMPARE_AMC.md — What AMC is and is not

AMC is strongest when compared honestly.

## AMC vs self-reported evals
Self-reported eval systems rely too heavily on what the agent or team claims.
AMC emphasizes execution evidence and weighted trust.

## AMC vs keyword scanners
Keyword scanners can detect patterns.
They cannot substitute for a trust system grounded in observed agent behavior.

## AMC vs prompt-security-only tools
Prompt injection testing matters, but it is only one slice of agent trust.
AMC spans scoring, assurance, governance, monitoring, compliance, and evidence integrity.

## Core idea
AMC is not just asking “did a prompt attack work?”
It is asking:
- what did the agent actually do?
- what evidence exists?
- how much trust should that evidence carry?
- what should be fixed next?
