# COMPATIBILITY_OVERVIEW.md — Where AMC fits

AMC should answer one question quickly:

> Will this work with my stack?

- CrewAI
- AutoGen
- OpenAI Agents SDK
- LlamaIndex
- Semantic Kernel

### Agent runtimes / tools
- Claude Code
- Gemini
- OpenClaw

- OpenAI-compatible endpoints

### Adoption paths
- browser exploration
- local CLI scoring
- CI gate
- self-hosted team deployment
- enterprise deployment

## Rule
Compatibility should be visible early on the homepage, README, and docs pathing — not buried deep in adapter docs.
