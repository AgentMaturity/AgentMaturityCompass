## Counter-Proposal Guide

*If the prospect pushes back on terms, use this framework:*

| They Want | Your Move |
|-----------|-----------|
| Lower price | Narrow scope to 3 critical dimensions (not recommended — explain why). Or: adjust payment to 3 installments instead of 2. Never discount below $4,000. |
| More deliverables | Scope as Phase 2 retainer add-on. Sprint scope is fixed for a reason — quality. |
| Longer timeline | Accommodate up to 7 business days if their evidence access is slow. No additional cost. |
| Pilot / trial | Sprint IS the pilot. Offer to start with one BU/team if enterprise. |
| Custom dimensions | Add 1-2 custom dimensions for $1,000 each. |
| Legal/procurement review | Expected for enterprise. Provide SOW, security questionnaire, insurance cert. Allow 5-10 business days but keep the champion warm. |
