### 1. Purpose

Provider will conduct the AMC Compass Sprint, a 5-business-day AI agent maturity assessment for Client, delivering the specific deliverables enumerated in Section 2.

### 2. Deliverables

Provider shall deliver the following:

| # | Deliverable | Acceptance Criteria |
|---|------------|-------------------|
| 2.1 | Maturity Baseline Scorecard | Scores across 7 dimensions (L1–L5), each backed by ≥1 evidence citation from Client's environment |
| 2.2 | Gap Analysis Report | Written report identifying gaps between current state, claimed state, and target state, with severity ratings (Critical / High / Medium / Low) |
| 2.3 | 30/60/90-Day Prioritized Roadmap | Action items sequenced by impact and feasibility, with suggested owners and dependencies |
| 2.4 | Evidence Vault Setup | Template and collection guide for ongoing evidence management |
| 2.5 | Executive Deck | 10-slide presentation suitable for board/leadership audience |
| 2.6 | Technical Appendix | Dimension-level scoring detail, methodology notes, and raw rationale |
| 2.7 | Readout Call | 30-minute video call walkthrough of findings with Q&A (recorded and provided to Client) |
