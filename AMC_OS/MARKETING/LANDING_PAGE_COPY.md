## COPY NOTES (for devs/designers)

- Hero animation: compass needle drifts chaotically → settles to true north on page load
- Scroll trigger: dimensions reveal evidence requirements on hover
- CTA button: subtle tactile micro-pulse; no flashy gradients
- Font pairing: Inter 400/600 (body) + DM Serif Display (headlines)
- Section dividers: thin ruled lines (like a technical drawing)
- Spacing system: 8px base grid, generous whitespace
- Mobile: single column, CTA always in viewport
- Accessibility: reduce-motion media query on all animations
- Performance: no heavy JS animations; CSS transforms only
