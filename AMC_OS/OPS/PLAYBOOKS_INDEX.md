## How to use
1. Every agent reads 00_GLOBAL_STANDARDS.md before starting
2. When writing copy: apply StoryBrand + Ogilvy
3. When doing sales: apply SPIN + Challenger + Mom Test
4. When building product: apply JTBD + Traction Bullseye
5. Before shipping anything: run Da Vinci QA Checklist
6. Every day: run KAIZEN loop (Skill Ledger + Impact Log + 1 experiment)
