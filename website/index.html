How It Works

Three steps to
verified trust.

01
Install & Connect
One command installs AMC. Connect your agent through the gateway — supports OpenAI, Anthropic, Gemini, Grok, and any OpenAI-compatible API.
npm i -g agent-maturity-compass && amc setup
02
Observe & Score
AMC silently observes agent behavior against 111 maturity criteria across 5 dimensions. Every observation is cryptographically signed and appended to a Merkle-tree ledger.
03
Prove & Improve
Get an evidence-backed maturity score (L0–L5). Export an Agent Passport — a portable, verifiable trust credential. Track improvement over time with visual dashboards.
