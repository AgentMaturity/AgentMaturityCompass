Product Family

One trust stack. Eight named surfaces.

Keep the names. Explain them faster. AMC is not one command with a fancy logo — it is a trust stack you can grow into.

Score

Evidence-weighted trust scoring, gap analysis, and maturity diagnostics from execution behavior instead of self-reported claims.

Shield

Adversarial assurance packs for prompt injection, exfiltration, context leakage, over-compliance, and other agent failure modes.

Enforce

Policy controls, approval workflows, scoped actions, and governance mechanisms for higher-trust execution environments.

Vault

Signatures, keys, tamper-evident ledgers, and proof infrastructure for scores, reports, and audit-grade evidence chains.

Watch

Traces, anomalies, timelines, monitoring, and operational drift detection so you can see what agents actually do over time.

Fleet

Compare, benchmark, inventory, and govern multiple agents, teams, and trust programs together instead of one-by-one chaos.

Passport

Portable identity, credential, and trust portability artifacts for agents that need continuity across environments and boundaries.

Comply

Regulatory mappings, audit binders, governance reports, and compliance outputs for frameworks like EU AI Act and ISO 42001.

FAQ

Common questions

When scored using keyword matching or self-reported documentation, an AI agent scores 100/100. When AMC evaluates the same agent using execution-verified evidence — watching what the agent actually does when safety controls are triggered — the real score is 16/100. That 84-point gap is documentation inflation. AMC closes it with cryptographic evidence chains.

Most frameworks let the agent being evaluated provide its own evidence — a structural conflict of interest. AMC sits between the agent and the LLM provider as a trusted observer, capturing every API call and tool use with Ed25519 signatures. Scores come from what the agent does, not what it claims. Self-reported evidence is capped at 0.4× weight; observed evidence scores at 1.0×.

L0: No safety controls — running with scissors. L1: Basic content filters, minimal logging. L2: Structured oversight, basic policy enforcement — where most production agents are today. L3: Consistent governance, full audit trails — EU AI Act minimum. L4: Cryptographic proof chains, statistical confidence. L5: Self-governing with evidence, continuous improvement with proof.
