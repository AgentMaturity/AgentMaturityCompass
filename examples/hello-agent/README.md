# Hello Agent — AMC Example

A minimal example showing how to score an AI agent with AMC in under 5 minutes.

## Prerequisites

- Node.js ≥ 20
- AMC installed: `npm i -g agent-maturity-compass`

## Quick Start

```bash
# 1. Set up your vault passphrase
export AMC_VAULT_PASSPHRASE='demo-passphrase'

# 2. Initialize the workspace
cd examples/hello-agent
amc init

# 3. Get a quick maturity score
amc quickscore

# 4. See the 84-point gap demo
amc demo gap

# 5. Generate guardrails for your agent
amc guide --go

# 6. Run the red-team assurance packs
amc assurance run --all

# 7. Check EU AI Act classification
amc quickscore --eu-ai-act
```

## What's Here

- `agent.js` — A trivial "agent" that echoes input (for demonstration)
- `AGENTS.md` — Will be auto-generated by `amc guide --go` with guardrails

## Scoring a Real Agent

To score your actual agent, wrap it with AMC's evidence capture:

```bash
# Wrap any CLI agent
amc wrap generic-cli -- node agent.js "What is 2+2?"

# Or wrap a LangChain agent
amc wrap langchain -- python my_agent.py

# Then re-score with captured evidence
amc quickscore
```
